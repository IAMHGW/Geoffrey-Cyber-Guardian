# 📱 Geoffrey Mobile App Installation Guide

## Install Geoffrey on Your Mobile Device - Ready to Roll!

Geoffrey v5.0 Enterprise is now available as a **Progressive Web App (PWA)** that works on both iOS and Android devices!

---

## 📲 INSTALLATION INSTRUCTIONS

### 🍎 iPhone/iPad (iOS) Installation:

1. **Open Safari** (must use Safari, not Chrome)
2. Navigate to: **https://iamhgw.github.io/Geoffrey-Cyber-Guardian/**
3. Wait for the page to load completely
4. Tap the **Share button** at the bottom (square with arrow pointing up)
5. Scroll down and tap **"Add to Home Screen"**
6. (Optional) Edit the name if desired
7. Tap **"Add"** in the top right corner
8. 🎉 **Done!** Geoffrey icon appears on your home screen

**iOS Features:**
- Full-screen experience (no Safari UI)
- Works like a native app
- Fast launch from home screen
- Offline capability
- Push notifications (future)

---

### 🤖 Android Installation:

**Method 1: Chrome (Recommended)**
1. **Open Chrome browser**
2. Navigate to: **https://iamhgw.github.io/Geoffrey-Cyber-Guardian/**
3. Tap the **menu icon (⋮)** in the top right
4. Select **"Install app"** or **"Add to Home Screen"**
5. Tap **"Install"** when prompted
6. 🎉 **Done!** Geoffrey icon appears on your home screen

**Method 2: Chrome Install Banner**
1. Visit the site in Chrome
2. Wait for the **"Add to Home Screen"** banner to appear
3. Tap **"Add"**
4. Follow the prompts

**Android Features:**
- Standalone app experience
- No browser address bar
- Fast performance
- Background updates
- Notifications support

## 🔧 TROUBLESHOOTING

### iOS Issues:

**"Add to Home Screen" option not showing?**
- Make sure you're using Safari (not Chrome/Firefox)
- Reload the page completely
- Update iOS to latest version

**App not opening in full screen?**
- Delete the app from home screen
- Reinstall following steps above
- Make sure PWA meta tags are present

### Android Issues:

**"Install app" option not showing?**
- Use Chrome browser (not Firefox/Opera)
- Update Chrome to latest version
- Clear browser cache and try again

**App icon not appearing?**
- Check home screen and app drawer
- Restart device
- Reinstall the app
